EsquemaPArticionado
===================================
EJERCICIOS GUIADOS
	-En sistemas Linux, ¿dónde se almacenan los archivos para el gestor de arranque GRUB?
		En la particion de arranque /boot/grub

	-¿Dónde debe terminar la partición de arranque para garantizar que una PC siempre pueda cargar el kernel?
		desde los 528 MB o antes del cilindr 1024

	-¿Dónde se suele montar la partición EFI?
		en la particion /boot/efi

	-Al montar manualmente un sistema de archivos, ¿en qué directorio se debe montar normalmente?
		/mnt. sin embargo no es obligatorios, lo puede montar en donde quiera


EJERCICIOS EXPLORATORIOS
	-¿Cuál es la unidad más pequeña dentro de un grupo de volúmenes?
		Los grupos de volúmenes se subdividen en extensiones (extends).

	-¿Cómo se define el tamaño de un volumen lógico?
		Por el tamaño de las extensiones físicas multiplicado por el número de extensiones en el volumen.

	-En un disco formateado con el esquema de partición MBR, ¿cuál es el ID de la partición del sistema EFI?
		0xEF

	-Además de las particiones de intercambio, ¿cómo puede aumentar rápidamente el espacio de intercambio en un sistema Linux?
		Se puede usar archivos de intercambio

EJERCICIOS GUIADOS

	-¿Cuál es la ubicación predeterminada para el archivo de configuración GRUB 2?
		/boot/grub/grub.cfg

	-¿Cuáles son los pasos necesarios para cambiar la configuración de GRUB 2?
		modificar el archivo mencionado en la anterior pregunta y despues ejecutar update-grub2

	-¿En qué archivo se deben agregar entradas de menú personalizadas de GRUB 2?
		en el arhcivo /etc/grub.d/40_custom

	-¿Dónde se almacenan las entradas de menú para GRUB Legacy?
		en /boot/grub/menu.lst

	-Desde un menú GRUB 2 o GRUB Legacy, ¿cómo puede ingresar a la consola de GRUB?
		apretando c 


EJERCICIOS EXPLORATORIOS 
	-Imagine un usuario que configura GRUB Legacy para arrancar desde la segunda partición del primer disco. Escribe la siguiente entrada de menú personalizada:
	title My Linux Distro
	root (hd0,2)
	kernel /vmlinuz root=/dev/hda1
	initrd /initrd.img
	Sin embargo, el sistema no se iniciará. ¿Qué está mal?
		La particion, en legacy, los discos se enumeran desde el 1, por lo que seria, root (hd1,2)

	-Imagine que tiene un disco identificado como /dev/sda con múltiples particiones. ¿Qué comando se puede usar para averiguar cuál es la partición de arranque en un sistema?
		con fdisk -l /dev/sda y la particion deberia de tener un *  

	-¿Qué comando se puede usar para averiguar el UUID de una partición?
		blkid o ls -l /dev/disk/by-uuid/

	-Considere la siguiente entrada para GRUB 2
	menuentry "Default OS" {
	    set root=(hd0,1)
	    linux /vmlinuz root=/dev/sda1 ro quiet splash
	    initrd /initrd.img
	}
	Cámbiela para que el sistema arranque desde un disco con el UUID 5dda0af3-c995-481a-a6f3-46dcd3b6998d
		Se debe de agregar la linea, --no-floppy --set-=root --fs-uuid 5dda0af3-c995-481a-a6f3-46dcd3b6998d

	¿Cómo puede configurar GRUB 2 para que espere 10 segundos antes de iniciar la entrada de menú predeterminada?
		con la opcion del grub, GRUB_TIMEOUT=10

	Desde una consola GRUB Legacy, ¿cuáles son los comandos para instalar GRUB en la primera partición del segundo disco?
		grub> root (hd1,0)
	grub> setup (hd1)

Gestion de paquetes Debian
============================
EJERCICIOS GUIADOS
	-¿Cuál es el comando para instalar un paquete llamado package.deb usando dpkg?
		dpkg -i package.deb

	-Usando dpkg-query, encuentre qué paquete contiene un archivo llamado 7zr.1.gz.
		dpkg-query -S 7zr.1.gz	

	-¿Puede eliminar un paquete llamado unzip del sistema usando dpkg -r unzip si el paquete file-roller depende de él? Si no, ¿cuál sería la forma correcta de hacerlo?
		No. dpkg no resolverá las dependencias y no le permitirá eliminar un paquete si otro paquete instalado depende de él. En este ejemplo, primero puede eliminar file-roller (suponiendo que nada depende de él) y luego eliminar unzip, o eliminar ambos al mismo tiempo con:
		# dpkg -r unzip file-roller

	-Usando apt-file, ¿cómo puede averiguar qué paquete contiene el archivo unrar?
	apt-file search unrar 

	-Usando apt-cache, ¿cuál es el comando para mostrar información para el paquete gimp?
		apt-cache show gimp


EJERCICIOS EXPLORATORIOS
	-Considere un repositorio con paquetes fuente de Debian para la distribución xenial, alojado en http://us.archive.ubuntu.com/ubuntu/ y con paquetes para el componente universe. ¿Cuál sería la línea correspondiente que se agregará a /etc/apt/sources.list?
		deb-src http://us.archive.ubuntu.com/ubuntu/ xential universe

	-Mientras compila un programa, se encuentra con un mensaje de error indicándole que el archivo de cabecera zzip-io.h no está presente en su sistema. ¿Cómo puede averiguar qué paquete proporciona ese archivo?
		sudo apt-file search zzip-io.h

	-¿Cómo puede ignorar una advertencia de dependencia y eliminar un paquete usando dpkg, incluso si hay paquetes que dependen de él en el sistema?
			dpkg -f -r paquete  o con --force

	-¿Cómo puede obtener más información sobre un paquete llamado midori usando apt?
		sudo apt-cache show midori

	-Antes de instalar o actualizar paquetes con apt, ¿qué comando se debe usar para garantizar que el índice de paquetes esté actualizado?
		aot-get update, Esto descargará los últimos índices de paquetes de los repositorios descritos en el archivo /etc/apt/sources.list o en el directorio /etc/apt/sources.list.d/.

Gestion de paquetes RPM y YUM
==============================
EJERCICIOS GUIADOS
	-Usando rpm en un sistema Red Hat Enterprise Linux, ¿cómo instalaría el paquete file-roller-3.28.1-2.el7.x86_64.rpm mostrando una barra de progreso durante la instalación?
		rpm -ihv file-roller-3.28.1-2.el7.x86_64.rpm

	-Usando rpm, descubra qué paquete contiene el archivo /etc/redhat-release.
		rpm -qf /etc/redhat-release

	-¿Cómo usaría yum para buscar actualizaciones para todos los paquetes en el sistema?
		yum check-update

	-Usando zypper, ¿cómo deshabilitaría un repositorio llamado repo-extras?
		zypper modifyrepo -d repo-extras

	-Si tiene un archivo .repo que describe un nuevo repositorio, ¿dónde se debe colocar este archivo para que DNF lo reconozca?
		en la ruta /etc/yum.repos.d/

EJERCICIOS EXPLORATORIOS
	-¿Cómo usaría zypper para averiguar qué paquete posee el archivo /usr/sbin/swapon?
		zypper search --provides /usr/sbin/swapon

	-¿Cómo puede obtener una lista de todos los paquetes instalados en el sistema usando dnf?
		dnf list -installed 

	-Usando dnf, ¿cuál es el comando para agregar un repositorio ubicado en https://www.example.url/home:reponame.repo al sistema?
		Trabajar con repositorios es un “configuration change”, así que utilice el parámetro config-manager y el parámetro --add_repo: dnf config-manager --add_repo https://www.example.url/home:reponame.repo.

	-¿Cómo puede usar zypper para verificar si el paquete unzip está instalado?
		Debe hacer una búsqueda (se) en los paquetes instalados (-i): zypper se -i unzip.

	-Usando yum, descubra qué paquete proporciona el archivo /bin/wget
		yum whatprovides /bin/wget
